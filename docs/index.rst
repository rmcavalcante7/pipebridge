PipeBridge Documentation
========================

PipeBridge is an unofficial Python SDK for Pipefy focused on typed models,
safe workflows, and extensible integrations.

It is designed for teams that need more than a thin GraphQL wrapper:

- typed models and semantic navigation helpers
- safe card update and move flows
- upload and download workflows with policies
- extensibility through rules, handlers, and custom steps
- reusable pipe schema cache
